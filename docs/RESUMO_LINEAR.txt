2. Modo Interativo

   ruby linear_cut_optimizer.rb -f exemplo_tubos.yml -j

4. Ajustar espessura de corte
   ruby linear_cut_optimizer.rb -f exemplo.yml -c 4

📊 FORMATO YAML (Bilíngue)

  - identificacao: "Tubo 30x30"  # label
    comprimento: 6000  # length (mm)
    quantidade: 5  # quantity

pecas_necessarias:  # required_pieces
  - identificacao: "Montante"  # label
    comprimento: 2100  # length (mm)
    quantidade: 4  # quantity

🎯 ALGORITMO FFD (First Fit Decreasing)

1. Ordena peças por comprimento (maior → menor)
2. Para cada peça:
   - Tenta colocar na primeira barra que couber
   - Se não couber, usa nova barra
3. Considera espessura de corte entre peças

Eficiência típica: 90-98% 🎉

💡 QUANDO USAR

Use Linear (1D) quando:
  ✅ Material tem apenas 1 dimensão relevante
  ✅ Tubos, barras, perfis, sarrafos
  ✅ Quer máxima eficiência
  ✅ Visualização simples é suficiente

Use 2D quando:
  ✅ Material tem 2 dimensões
  ✅ Chapas, placas, vidros
  ✅ Precisa importar CAD
  ✅ Quer visualização SVG
